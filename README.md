The art and audio companion mod for **Sosciencity**.

*Co-Authors:* Johanna Spieker, _traum

*Discord:* https://discord.gg/yg7yytYQRq

# Description

This mod stores the graphics and audio files for Sosciencity, separated from the main mod so players don't have to redownload all the assets after every code update.

It contains no game logic of its own - it is a required dependency of the Sosciencity mod and does nothing on its own.
